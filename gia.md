# como usar git colaborativo

## clonar repositorio
    git clone (link)
# ramas
## ver ramas locales
    git brach 
## ver ramas remotas
    git brach -r
## crear rama 
    git brach (nombre de la rama)
## eliminar rama
    git brach -d
## moverse entre ramas
    git swtch (nombre de la rama)
## subir rama al remoto
    git push origin (nombre de la rama)
