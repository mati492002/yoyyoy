# como usar ramas
## clonar repositorio
    git clone link
## creas la rama
    git brach nombre-de-la-rama
## elegis la rama
    git swtch nombre-de-la-rama
## subis los cambios
    git add .
## creas un commit
    git commi -m "nombre"
## subir al repositorio
    git push origin nombre-de-la-rama
## moverse ala rama master
    git swith master
## traer los cambios de la rama que queremos
    git merge nombre-de-la-rama
